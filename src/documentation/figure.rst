*******
Figure
*******

:file:`figure_maker.py` combines the predicted prices for generated houses and takes the mean percentage change monthly and yearly. It creates figures using that percentage change and TurkStat data explained in :ref:`data` section. It stores the the produced figures in **bld/out/figures**. In that folder also project paths of the project can be found, specifically dependencies are shown in the :file:`dag.png`
